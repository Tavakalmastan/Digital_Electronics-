# Day 21 – Practice Session (Sequential Circuits)

## Overview

Today's session was mainly focused on solving practice problems related to sequential circuits. Instead of learning new theory, I spent time implementing flip-flops using multiplexers, analyzing sequential circuits, solving timing diagrams, and designing sequence counters. This helped me understand how the concepts learned previously are applied in actual circuit design.

---

## 1. T Flip-Flop using 2:1 Multiplexer

Started by designing a T Flip-Flop using a 2:1 MUX and a D Flip-Flop. Since the complement of the T input was required, I also learned how a NOT gate can be implemented using another 2:1 MUX. After completing the connections, I verified that the circuit behaves like a standard T Flip-Flop, where the output toggles whenever T = 1 and remains unchanged when T = 0.

---

## 2. Active-Low Asynchronous Reset T Flip-Flop

Next, I modified the previous design by adding an active-low asynchronous reset. The reset input was connected through another 2:1 MUX.

Observed the behavior:

* Reset = 0 → Output is immediately cleared to 0, irrespective of the clock.
* Reset = 1 → Circuit behaves as a normal T Flip-Flop.

This exercise gave a better understanding of how asynchronous reset works in practical sequential circuits.

---

## 3. JK Flip-Flop using 2:1 Multiplexer

Designed a JK Flip-Flop using only multiplexers. This showed that even without using a dedicated JK Flip-Flop IC, the same functionality can be achieved by properly selecting inputs through multiplexers. It was a good exercise to understand the internal working of JK Flip-Flops.

---

## 4. Finding the Modulus of a Sequential Circuit

Solved a sequential circuit problem where the objective was to determine its modulus.

The steps followed were:

* Derived the input equations for all D Flip-Flops.
* Constructed the state table.
* Traced the output sequence.
* Identified the repeating states.

The derived equations were:

* D₀ = Q₂ + Q₁
* D₁ = Q̅₁
* D₂ = Q₂

From the repeating sequence, the modulus of the circuit was determined.

---

## 5. JK Flip-Flop Timing Diagram

Practiced a timing diagram problem where:

* J = Q̅
* K = 1

For every clock pulse, I traced the output manually and verified the sequence generated by the JK Flip-Flop. This improved my understanding of present state, next state, and edge-triggered operation.

---

## 6. Revision of Characteristic and Excitation Tables

Spent some time revising Characteristic Tables and Excitation Tables since they are frequently used while designing counters.

The revision included:

* D Flip-Flop
* JK Flip-Flop
* T Flip-Flop

This made it easier to derive required flip-flop inputs from state transitions.

---

## 7. Sequence Counter Design using D Flip-Flops

Designed a sequence counter for the sequence:

**1 → 2 → 5 → 6 → 1**

The complete design procedure included:

* Writing the present state table.
* Writing the next state table.
* Finding the required D inputs.
* Drawing K-Maps.
* Simplifying Boolean expressions.
* Implementing the final logic circuit.

This exercise helped me understand the complete design flow of a synchronous sequence counter.

---

## 8. Ripple Counter Timing Analysis

Finally, I solved a ripple counter timing problem using cascaded JK Flip-Flops.

I drew the timing waveforms for:

* Clock
* Q₁
* Q₂

From the waveforms, I observed how each flip-flop changes state only after the previous stage changes, which clearly demonstrates the ripple effect caused by propagation delay.
