# Day 13 - Practice Problems

## Problem 1: Find the Value of Base x

Given:

```text
(135)x + (144)8 = (214)x + 2
```

Expanding:

```text
(135)x = x² + 3x + 5

(144)8 = 64 + 32 + 4 = 100

(214)x = 2x² + x + 4
```

Substituting:

```text
x² + 3x + 105 = 2x² + x + 6

x² - 2x - 99 = 0

(x - 11)(x + 9) = 0
```

Result:

```text
x = 11
```

## Problem 3: Find the Base x

Given:

```text
(136)10 = (210)x
```

Expanding:

```text
136 = 2x² + x

2x² + x - 136 = 0
```

Using quadratic formula:

```text
x = [-1 ± √(1 + 4×2×136)] / 4

x = [-1 ± √1089] / 4

x = [-1 ± 33] / 4
```

Positive solution:

```text
x = 8
```

Result:

```text
(136)10 = (210)8
```
